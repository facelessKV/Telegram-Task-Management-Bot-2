# ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ЗАПУСКУ TELEGRAM БОТА ДЛЯ УПРАВЛЕНИЯ ЗАДАЧАМИ

## ДЛЯ ПОЛЬЗОВАТЕЛЕЙ WINDOWS

### Шаг 1: Установка Python
1. Откройте браузер и перейдите на сайт https://www.python.org/downloads/
2. Скачайте Python 3.9.13 (не последнюю версию, так как она может иметь проблемы с установкой зависимостей)
3. Запустите скачанный файл (например, python-3.9.13-amd64.exe)
4. Обязательно отметьте галочку "Add Python 3.9 to PATH" перед установкой!
5. Нажмите "Install Now" и дождитесь завершения установки

### Шаг 2: Скачивание и подготовка бота
1. Создайте на компьютере папку для бота, например: C:\TelegramTaskBot
2. Скопируйте файлы бота (main.py и config.py) в эту папку

### Шаг 3: Получение токена для бота
1. Откройте Telegram и найдите @BotFather
2. Напишите боту команду /newbot
3. Следуйте инструкциям: введите имя бота, а затем придумайте уникальное имя пользователя, которое должно заканчиваться на "bot"
4. После создания бота вы получите API-токен (длинная строка букв и цифр), скопируйте его

### Шаг 4: Настройка бота
1. Откройте файл config.py в любом текстовом редакторе (можно через блокнот)
2. Замените 'YOUR_BOT_TOKEN' на полученный токен (сохраняя кавычки)
3. Сохраните и закройте файл

### Шаг 5: Установка необходимых библиотек
1. Откройте командную строку Windows: нажмите Win+R, введите cmd и нажмите Enter
2. Перейдите в папку с ботом. Например, наберите: 
   ```
   cd C:\TelegramTaskBot
   ```
3. Установите необходимые библиотеки с помощью команды:
   ```
   pip install aiogram==3.0.0 apscheduler
   ```
4. Дождитесь завершения установки

### Шаг 6: Запуск бота
1. В командной строке, находясь в папке с ботом, введите:
   ```
   python main.py
   ```
2. Если всё установлено правильно, вы увидите сообщение о запуске бота
3. НЕ ЗАКРЫВАЙТЕ командную строку, пока бот работает!

### Шаг 7: Использование бота
1. Откройте Telegram и найдите своего бота по имени, которое вы задали
2. Нажмите кнопку "Start" или отправьте команду /start
3. Следуйте инструкциям бота для создания и управления задачами

## ДЛЯ ПОЛЬЗОВАТЕЛЕЙ LINUX

### Шаг 1: Установка Python
1. Откройте терминал (Ctrl+Alt+T в большинстве дистрибутивов)
2. Обновите пакеты:
   ```
   sudo apt update
   ```
3. Установите Python и pip:
   ```
   sudo apt install python3.9 python3-pip
   ```

### Шаг 2: Создание папки для бота
1. Создайте папку для бота:
   ```
   mkdir ~/TelegramTaskBot
   ```
2. Перейдите в эту папку:
   ```
   cd ~/TelegramTaskBot
   ```

### Шаг 3: Создание файлов бота
1. Создайте файл main.py:
   ```
   nano main.py
   ```
2. Скопируйте содержимое файла main.py (весь код бота) в открывшийся редактор
3. Сохраните файл: нажмите Ctrl+O, затем Enter, затем Ctrl+X для выхода
4. Создайте файл config.py:
   ```
   nano config.py
   ```
5. Введите:
   ```python
   BOT_TOKEN = 'YOUR_BOT_TOKEN'  # Здесь нужно будет заменить на ваш токен
   ```
6. Сохраните файл: Ctrl+O, Enter, Ctrl+X

### Шаг 4: Получение токена для бота
1. Откройте Telegram и найдите @BotFather
2. Напишите боту команду /newbot
3. Следуйте инструкциям: введите имя бота, а затем уникальное имя пользователя, которое должно заканчиваться на "bot"
4. После создания бота вы получите API-токен (длинная строка букв и цифр), скопируйте его

### Шаг 5: Настройка бота
1. Откройте файл config.py для редактирования:
   ```
   nano config.py
   ```
2. Замените 'YOUR_BOT_TOKEN' на полученный токен (сохраняя кавычки)
3. Сохраните файл: Ctrl+O, Enter, Ctrl+X

### Шаг 6: Установка необходимых библиотек
1. В терминале, находясь в папке с ботом, введите:
   ```
   pip3 install aiogram==3.0.0 apscheduler
   ```
2. Дождитесь завершения установки

### Шаг 7: Запуск бота
1. В терминале, находясь в папке с ботом, введите:
   ```
   python3 main.py
   ```
2. Если всё установлено правильно, вы увидите сообщение о запуске бота
3. НЕ ЗАКРЫВАЙТЕ терминал, пока бот работает!

### Шаг 8: Использование бота
1. Откройте Telegram и найдите своего бота по имени, которое вы задали
2. Нажмите кнопку "Start" или отправьте команду /start
3. Следуйте инструкциям бота для создания и управления задачами

## ОСНОВНЫЕ КОМАНДЫ БОТА

- /start - Запустить бота и получить приветственное сообщение
- /add_task - Добавить новую задачу
- /list_tasks - Просмотреть список задач
- /update_task - Обновить существующую задачу
- /complete_task - Отметить задачу как выполненную

## РЕШЕНИЕ ПРОБЛЕМ

### Если бот не запускается:
1. Убедитесь, что вы правильно ввели токен в файле config.py
2. Проверьте, что установлены все необходимые библиотеки
3. Проверьте версию Python: введите в терминале или командной строке `python --version` или `python3 --version`

### Если не удаётся установить библиотеки:
1. Попробуйте использовать другую версию Python (3.8 или 3.9)
2. Проверьте соединение с интернетом
3. В Windows может потребоваться запустить командную строку от имени администратора

### Если бот запускается, но не отвечает в Telegram:
1. Проверьте, что вы общаетесь с правильным ботом (по имени пользователя)
2. Убедитесь, что командная строка или терминал, где запущен бот, все еще работает
3. Проверьте, нет ли ошибок в консоли, где запущен бот

## Для запуска бота в фоновом режиме (только Linux)

Чтобы бот продолжал работать после закрытия терминала:

1. Установите screen:
   ```
   sudo apt install screen
   ```
2. Создайте новую screen-сессию:
   ```
   screen -S taskbot
   ```
3. Запустите бота как обычно:
   ```
   python3 main.py
   ```
4. Нажмите Ctrl+A, затем D для отключения от сессии (бот продолжит работать)
5. Чтобы вернуться к боту позже, введите:
   ```
   screen -r taskbot
   ```
